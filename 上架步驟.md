# 今日報報 — 上架給同仁 步驟

## 目標網址（部署後）
https://jennyih0227-stack.github.io/today-report/

---

## 一、建立 repo 並上傳（約 5 分鐘）

1. 登入 GitHub 帳號 `jennyih0227-stack`
2. 右上角「+」→ New repository
3. Repository name 填 **today-report**
4. 選 **Public**（公開，同仁才能開）
5. 按 Create repository
6. 進到空 repo 頁 →「uploading an existing file」
7. 把本資料夾**所有檔案**拖進去上傳，包含：
   - index.html
   - README.md
   - 使用說明.png
   - data/ 資料夾（twii.json）
   - .github/ 資料夾（workflows/fetch-twii.yml）
   > 用網頁拖曳若無法上傳資料夾，改用手機/電腦的 GitHub App，
   > 或直接把整包 zip 解壓後逐一上傳。
8. 下方 Commit changes

## 二、開啟 GitHub Pages（讓網址生效）

1. repo → Settings → Pages
2. Source 選 **Deploy from a branch**
3. Branch 選 **main**、資料夾 **/(root)** → Save
4. 等 1～2 分鐘，網址就會生效

## 三、開啟台股自動更新

1. repo → Settings → Actions → General
2. 拉到「Workflow permissions」選 **Read and write permissions** → Save
3. 回 repo 上方 **Actions** 分頁 → 左邊「抓取台股加權指數」→ 右上 **Run workflow** 跑一次
4. 確認 data/twii.json 有更新即完成

---

## 四、發給同仁

- 把 **使用說明.png** 貼到 LINE 群組（圖上已含 QR 和網址）
- 或直接貼網址：https://jennyih0227-stack.github.io/today-report/
- 提醒同仁用 **Safari 或 Chrome** 開，下載最順

---

## 常見問題

**Q：同仁說打不開？**
A：確認 Pages 已 Save 且等過 1～2 分鐘；網址結尾的 `/` 不要漏。

**Q：台股沒數字？**
A：Actions 權限要設 Read and write 並手動 Run 過一次；
   或請同仁在工具裡點鉛筆鈕手動填。

**Q：匯率跟台銀牌告有一點點差？**
A：正常，匯率是市場中價；下載前可點鉛筆鈕微調對齊。
